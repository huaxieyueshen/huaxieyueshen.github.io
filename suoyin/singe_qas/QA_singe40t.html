非LODOP的各种现象

  • 一点打印，打印机驱动消失，Print Spooler关闭
  • Print Spooler服务，错误1053：服务没有及时响应或控制请求
  • 提示print driver host for 32bit applications 已停止工作
  • 队列里提示，错误-正在打印
  • c-lodop程序未响应

1.一点打印，打印机驱动消失，Print Spooler关闭

如果是所有打印机驱动都消失了，可能和系统打印服务(Print Spooler)有关。
百度下Print Spooler自动关闭问题，排查是否是操作系统问题。

2.排查windows服务和打印机

查看操作系统是否存在打印机，系统打印服务(Print Spooler)是否开启了

查看操作系统是否存在打印机，系统打印服务(Print Spooler)是否开启了，是否有共享打印机。
如果有共享打印机，删除后重新卸载试试。目前发现共享打印机故障可能导致安装问题。

安装一个虚拟打印机测试下，可以安装群文件的pdf虚拟打印机。
操作系统的设备和打印机界面有打印机后，再试试。
之前发现系统没有打印机，可能对windows打印服务有影响。

3.提示print driver host for 32bit applications 已停止工作

用xps或pdf虚拟打印机测试下，
如果虚拟打印机正常，说明和打印机或打印机驱动有关。百度一下这个提示print driver host for applications，参考网上的方法处理试试，重装该真实打印机驱动试试。
