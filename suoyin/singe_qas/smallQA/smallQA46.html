不同电脑不同排查

询问输出方式
打印项是什么语句输出的，是add_print_text纯文本，还是add_print_htm等超文本
询问哪里不同
是内容样式不同，还是打印位置不同偏移了。
虚拟打印机预览对比
在样式有问题的电脑上用该真实打印机和虚拟打印机预览对比下。

预览是打印的反显，观察预览效果，和下方状态栏的纸张是否一致和是否有可打区域的影响。
超文本输出，有的电脑样式不同
在该有问题的电脑上用虚拟打印机测试下，win7以后系统自带Microsoft XPS Document Writer虚拟打印机 或者 下载安装群文件里的Doro PDF Writer虚拟打印机 用虚拟打印机测试下

如果虚拟打印机也有问题，对照虚拟打印机和真实打印机预览，是否该打印机不支持自定义纸张或有可打区域影响了样式。
如果虚拟打印机也有问题，排查打印项的内容代码，排查下面的条目。

查看lodop内部解析的html信息，见http://www.c-lodop.com/faq/pp8.html 分析差异点，因浏览器版本不同遵循的html标准不同，造成某些标签属性显示有差异，尽量避免使用有差异的属性，可以在ie下不同版本仿真情况下验证差异 排查样式，尽量使用使用各浏览器无歧义的通用的 html css js，不支持的样式可尝试用其他样式代替。

可尝试把ADD_PRINT_HTM 换成 ADD_PRINT_HTML（图形模式）试试，ADD_PRINT_HTML不如ADD_PRINT_HTM清晰，适合高精度的打印机，两者各有优势。

Lodop是基于本机windows底层引擎进行渲染解析的，与ie调用的引擎一致。在样式有问题所在电脑上，尝试重置ie，升级ie， 或排查下样式http://www.c-lodop.com/faq/pp8.html，通过删减或做简单例子排查，找到出问题的样式，修改样式。
不想用虚拟打印机测试
样式不同的电脑，先都用虚拟打印机测试下，排除打印机的影响。
打印机不支持自定义纸张或有可打区域，也可能会影响样式。

如果虚拟打印机也有问题，对照虚拟打印机和真实打印机预览，是否该打印机不支持自定义纸张或有可打区域影响了样式。
如果虚拟打印机也有问题，排查打印项的内容代码或电脑本身设置了缩放等问题。
有问题的电脑上是否有缩放现象
在有问题的电脑上观察是否有缩放现象，这两个比例是否在100%
1.查看本机ie浏览器的缩放比例应在100%，
2.操作系统控制面板设置显示应在100%。
虚拟打印机也有问题，且电脑无缩放设置，用的超文本
如果在样式有问题的电脑上虚拟打印机输出也有问题，可排除打印机问题。
电脑缩放比例也正常，可排除电脑或ie缩放问题。
且输出的是超文本打印项，查看lodop内部解析的html信息，见http://www.c-lodop.com/faq/pp8.html，排查超文本内容代码。
已发超文本排查的，排查上面的条目
