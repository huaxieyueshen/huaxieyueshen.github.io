国产linux安装方法和报错

如何安装linux版本
参考http://www.c-lodop.com/download.html
linux版
根据CPU型号安装对应的linux版安装程序
终端里运行 sudo ./lodop install
核对cpu型号（发一下型号，飞腾）
CPU是什么型号的，根据对应的型号安装对应的linux版安装程序

CPU型号发一下，如果是ARM架构的CPU，可以安装这个

飞腾的，是ARM架构的，可以安装这个
cpu型号相关，暂不支持龙芯
目前仅支持x86 64架构和arm架构的cpu，mips64架构的正在研发中

国产CPU（支持第一类）：
Linux操作系统CPU架构分类：
第一类：x86或amd64架构：
支持中国海光、上海兆芯、美国Intel、AMD

第二类：mips64架构：
支持中科院龙芯、北京君正

第三类：arm64架构：
支持华为鲲鹏(920)、国科大飞腾(2000/2500)、中标麒麟

龙芯CPU是mips64架构，目前不支持，还在研发中
飞腾的如何安装
飞腾的安装带ARM字样的lodop7，http://www.c-lodop.com/download.html
解压缩后 终端里运行 sudo ./lodop install

麒麟990的CPU无法安装（或其他）
终端里运行下xhost si:localuser:root

然后运行 sudo ./lodop install


执行sudo那个，提示：没有那个文件或目录
sudo后面加空格，每个英文命令之间用空格间隔

sudo ./lodop install
安装成功了，访问欢迎页面，修改lodopFuncs.js
访问http://localhost:8000欢迎页面试试

说明安装启动成功正常，
默认lodopFuncs.js（发行包里的）是windows提示下载安装的，参考官网样例的lodopFuncs.js，
http://www.c-lodop.com/demolist/LodopFuncs.js

http://www.c-lodop.com/demolist/LodopFuncs.js里有判断linux版本命令
发行包里自带的lodoFuncs.js里都是windows的，您可以参考修改下
