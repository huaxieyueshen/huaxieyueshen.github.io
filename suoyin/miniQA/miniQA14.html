【LODOP】提示Printer selected is not valid

这个提示的英文部分是windows系统抛出的异常信息，意思是“所选择的打印机无效”！

1.安装最新版c-lodop测试下，http://www.c-lodop.com/download.html
2.系统打印服务(Print Spooler)是否开启了
3.查看操作系统是否存在打印机
4.查看操作系统是否有共享打印机，删除所有共享打印机再测试下
最常见的原因：windows系统默认打印机是一台网络共享打印机且该打印机目前不可访问。例如：网络断开原因、本地重启电脑后共享打印机没有记住共享端的用户名和密码等。

5.重装打印机驱动
用虚拟打印机测试下，如果虚拟打印机不提示这个，和打印机或打印机驱动或打印机对一些语句的支持有关，重装打印机驱动测试下
如果是刚刚添加打印机出现该问题，建议重新启动电脑（个别驱动程序要求系统重启打印服务）

6.去掉纸张语句测试下set_print_pagesize语句
是否调用了set_print_pagesize语句， 该语句可动态设置纸张大小，但目前发现有很多打印机不支持该操作，会表现出以上异常。 建议暂时去掉这个语句试试

7.类似的提示
There is no default printer currently selected
Operation not supported on selected printer

是否没有指定默认打印机，查看一下windows打印机列表，指定一下windows默认打印机。
