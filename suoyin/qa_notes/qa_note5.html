【非常见问题杂笔记】不支持拖动表格调整列宽等

1.打印设计的这个图标有什么作用吗，看起来像插表格的，但是点击不了,期望能可视化编辑表格，拖动调整行高列宽合并单元格之类
那个图标是纯文本的颜色，插入一个add_print_text，可以该更颜色。超文本呈现灰色不可点击状态，超文本不能通过这个设置颜色。
不支持直接插入表格。不支持拖动合并单元格指令的。
超文本表格只能通过css样式控制，参考样例10。 http://www.c-lodop.com/demolist/PrintSample10.html

2.代码没有设置缩放，为什么就一单打印出来缩小了，其它的单又正常的
是某个任务吗，是所有电脑都这样，还是只有一台电脑。

如果是所有电脑，某个任务，可能和该任务有关，
排查内部的html信息和样式。

如果是单个电脑：
排查ie浏览器，
和操作系统显示里，
两个地方是否设置了缩放。需要是100%

4.背景图片怎么打印不出来
css背景图，http协议会按异步方式下载背景图，所以很容易等不到下载完毕就开始打印了，故lodop不打印背景图。至于普通图片参考样例37 http://www.c-lodop.com/demolist/PrintSample37.html
用普通图片的输出方式。

是一个base64编码的图片，普通图片不涉及
background:url()这是背景图，这种都不行。不论是base64还是普通的url。

css背景图，参考http://www.c-lodop.com/faq/pp14.html
通常情况下打印图片都支持，因超文本背景图片是异步资源，所以默认是不打印的。
实现超文本包含背景图片效果可以有3种解决方法：
1.把ADD_PRINT_HTM 改为 ADD_PRINT_HTML ,图形模式输出；
2.使用透明图片效果实现，参考样例37；
3.用lodop函数语句输出背景图，参考样例9。
Lodop背景图ADD_PRINT_SETUP_BKIMG，参考http://www.c-lodop.com/demolist/PrintSample9.html

5.页眉页脚和下方内容有间距
中间的打印项调整下定位。调整top值。

这两部分是两个打印项吗，如果是两个打印项，只能通过第二个打印项的top值定位修改，缩小两个打印项的距离。

两个打印项之间的有间距。
在打印设计里拖动第二个打印项向上移动一点试试。

用打印维护调整吗
打印维护是针对单个客户端的，如果是所有客户端都有问题，可直接调整代码。
如果只有这个客户端端有间距，可以用打印维护。

6.表单里的数据过长，就会导致有几条数据显示异常，像第二条数据那样超出表单的边框。用的是缅甸语，中文和英文过长就不会有问题
如果是超文本，用css样式控制试试，加载css样式，参考样例10 http://www.c-lodop.com/demolist/PrintSample10.html

Lodop是基于本机windows底层引擎进行渲染解析的，与ie调用的引擎一致。lodop自己不带超文本解析引擎。

查看lodop内部解析的html信息，见http://www.c-lodop.com/faq/pp8.html
分析差异点，因浏览器版本不同遵循的html标准不同，造成某些标签属性显示有差异，尽量避免使用有差异的属性，可以在ie下不同版本仿真情况下验证差异
排查样式，尽量使用使用各浏览器无歧义的通用的 html css js，不支持的样式可尝试用其他样式代替。

可尝试把ADD_PRINT_HTM 换成 ADD_PRINT_HTML（图形模式）试试，ADD_PRINT_HTML不如ADD_PRINT_HTM清晰，适合高精度的打印机，两者各有优势。

查看lodop内部解析的html信息，见http://www.c-lodop.com/faq/pp8.html
把拷出来的内容，放到ie里查看下。需要在ie里调整到合适的效果。

8.LODOP.ADD_PRINT_TEXT 怎么让这个text 一直处于（上面是ADD_PRINT_TABLE）最下面

一个内容关联别人后，其Top值不再是上边距，而是与被关联者的间隙距离,Left值也变为左边距相对偏离量。

linkedItem 后面的1 指的哪个
