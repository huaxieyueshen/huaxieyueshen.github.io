Lodop打印设计（PRINT_DESIGN）里的快捷键

Lodop打印控件，给开发人员提供了可视化编辑工具，编辑后可生成代码，然后复制到自己程序代码中，让开发更简单，打印设计中有一些快捷键也能让开发更便捷。

打印设计快捷键：
ctrl 多个选中
Shift 左侧打印项列表，连续一段选择
ctrl+鼠标滑动拖拉选中区域内打印项
按住alt+打印对象，左边打印项列表中鼠标拖动改变打印项的输出顺序
ctrl + delete 删除
ctrl + 方向键移动
按住ctrl+打印对象，同时拖拉一个对象就复制了

详细介绍和图示：
ctrl 多个选中
和windows系统选择多个不连续的文件一样，按住ctrl，点击多个不连续的文件，可以都选中，选中多个，在lodop中，可以在界面选，也可以在左侧的列表里选。
（点击左侧的小三角形可以展开元素列表，或者打印项的列表）

Shift 左侧打印项列表，连续一段选择
和windows连续选择文件一样,按住shift，选择起始项，就可以连续选择，在lodop中，可以这种方式选择左侧的列表，通过连续选择列表选中右侧界面上的元素。
由于界面上的元素是平面上的坐标，所以没法这样连续选择。

ctrl+鼠标滑动拖拉选中区域内打印项
在lodop中打印设计中，按住ctrl拖动鼠标左键，可以框选，框选方法和ps的矩形选框、qq截图等很多软件的框选方法相同，矩形选框。
上面的shift连续选择只能是左侧的列表，这个框选是右侧的打印设计的界面，框选可以选择多个平面上的元素。

按住alt+打印对象，左边打印项列表中鼠标拖动改变打印项的输出顺序
在lodop打印设计中该表列表的顺序，也改变了打印项的输出顺序，列表的顺序就是生成代码里面每个元素输出语句的顺序，一次只能拖动改变一个打印项的顺序。
选中多个，是不能同时拖动多个打印项改变顺序的。

ctrl + delete 删除
删除可以用ctrl+delete，选中元素（列表选，界面选，框选等），然后用这个组合快捷键，就可以快捷的删掉一个或多个元素。

ctrl + 方向键移动
微调可以在打印设计上方栏里，点击表示上下左右的小三角进行每次移动1px的微调，小三角的尖端部分就是移动方向，
选中元素，使用快捷键，按住ctrl+方向键，也可以快速微调。

按住ctrl+打印对象，同时拖拉一个对象就复制了
可以一个多次复制，多个多次复制等等，选中的元素可以复制到拖动到的位置。
