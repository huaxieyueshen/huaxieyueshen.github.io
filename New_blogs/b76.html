Win10图片打不开文件系统错误2147416359解决方法

该问题表现为win10打开所有图片都会提示这个‘文件系统错误-2147416359’，打开其他文件没问题。
此问题应该是win10自带的图片查看器出了故障。

在网上找到如下方案，但是我的服务列表里没有这个，只能另找方案。
在服务里找到windows license manager service服务，右键打开属性---启动类型改为自动---重启生效。

此问题为：
1.win10操作系统，打开所有图片都会提示这个‘文件系统错误-2147416359’
2.服务里找不到windows license manager service

出现原因不明，可能是没正常关机导致的，以前都是关闭大多数应用后关机，由于中午走得急，就那一次很多应用没有关，直接点关机，下午上班出现了这个错误，可能是由于关机时自动强制关闭一些应用导致的。

此问题可能是win10默认图片查看器不知道出了什么问题导致，在服务找不到windows license manager service的情况下，可以装一下其他图片查看器软件用来代替，安装后需要指定为默认查看图片工具，否则双击普通打开还是会提示这个，无法查看。

win10开始-设置-应用-默认应用-照片-选择另一个图片查看器（默认另一个图片查看器）
